__authenticate(browser, authOptions, pageLink) → {Promise.<*>}

Source:
  • main/validator.js, line 581
Use this function to load your own authentication handler. In your definition file, add the "auth" option. See the example.
Examples

my-definition.js

links:
     - link: 'https://www.nav.no'
       options:
         auth:
           type: 'isso'
           env: 'test'
           handler: './my-auth-handler'
         commands:
         - waitFor: 'body'

my-auth-handler.js

exports.handleAuthentication = function(browser, options) {
    // implement your own handler here
}
